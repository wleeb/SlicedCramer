

                    *** Installation ***


- To compile the C code, run the script ``cdfinvs''.
This creates the shared object file cdfinvs.so
(which is callable from within Python using the ctypes package).

- The user should install the FINUFFT Python package, available from
https://finufft.readthedocs.io/en/latest/

- The test code can be run using the script ``cramer'',
which calls the function main() from within cramer.py.

- Documentation on the relevant functions in the Python file
is contained within the file cramer.py, after the test code.
Documentation for each user-callable function is contained
after the function declaration.
